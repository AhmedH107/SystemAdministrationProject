# SystemAdministrationProject

A Linux system administration project focused on configuring, documenting, and automating common server administration tasks.

This project includes setup notes, scripts, and documentation for working with networking, DNS, LDAP, NTP, package management, storage, virtual machines, and system configuration in a Linux environment.

## Features

- Linux system administration documentation
- Shell and Python scripting
- Network configuration
- DNS setup and troubleshooting
- LDAP user and directory management
- NTP/time synchronization configuration
- Storage and filesystem-related setup
- QEMU/virtual machine environment notes
- Package management using APT
- Service management and command-line administration

## Tech Stack

- Linux
- Shell scripting
- Python
- Bash
- APT
- DNS
- LDAP
- NTP
- QEMU
